## Incubyte Salary Management System

### Tech Stack
- Java 25
- Spring Boot
- SQLite
- JUnit 5

### Features
- Employee CRUD
- Salary calculation with country-based deductions
- Salary metrics by country and job title

### TDD Approach
All features were developed using strict Test-Driven Development.

### AI Usage
AI (ChatGPT) was used for:
- Initial project scaffolding
- Test case generation
- API contract drafting
- Refactoring suggestions

All business logic, validations, and trade-offs were manually reviewed and refined.

### Running the App
mvn clean install
mvn spring-boot:run
